## Exercises

Once you have claude code running locally, or via codespaces, try the following tasks:


- what is the VBO ID for an ocicat?
- what is the VBO ID for an abyssinian?
- what are some cattle breeds?
- what are some synonyms of Domestic Shorthair cats?
- how many terms are there in the ontology?
- check all sheep breeds in VBO - are there any duplicates?

Editing tasks:

- add a test synonym
- Add the cat venom mappings and missing breeds from: https://gist.github.com/cmungall/fdf549c3e6c3ca88f4478ba0dc768c5d

Bonus:

- create a branch (make sure it is unique - include your name)
- ask the agent to make a PR
